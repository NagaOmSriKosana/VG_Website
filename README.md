# VyomGarud — Web Developer Intern Assessment

A single-page React landing site for VyomGarud (UAV systems). Built with Create React App.

## Features
- Dark, military-futuristic aesthetic (charcoal + orange #ff7b00)
- Hero, About, Products, Highlights, Contact & Footer sections
- Responsive layout (desktop/tablet/mobile)
- Subtle CSS animations; optional Framer Motion support
- Demo form (client-side) — replace with real backend for leads

## Stack
- React (Create React App)
- Plain CSS (BEM-like, modular styles)
- Optional: framer-motion for animations
